## 📌 Project Overview

This project is a **full-stack E-Commerce Website** integrated with a powerful **Admin Panel** that allows administrators to manage the entire online store from a centralized dashboard.

The system provides a seamless shopping experience for users while offering advanced management capabilities for administrators.

---

## 🚀 Features

### 👤 User Side

* User Registration & Login Authentication
* Browse Products
* Product Details View
* Add to Cart
* Order Placement
* Responsive UI Design

### 🛠 Admin Panel

* Admin Authentication
* Add / Edit / Delete Products
* Manage Orders
* Manage Users
* Dashboard Analytics
* Inventory Management

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/ANIQUESK/E-COMMERCE-WEBSITE-WITH-ADMIN-PANEL.git
cd E-COMMERCE-WEBSITE-WITH-ADMIN-PANEL
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Start Backend Server

```bash
npm run server
```

### 4️⃣ Start Frontend

```bash
npm start
```
